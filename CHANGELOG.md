# Smart Photo Toolkit v37.2 Alpha

## Added
- Document Studio now supports downloaded document PDF files and image files.
- Universal drag-select crop system for Aadhaar, Voter ID, PAN, Ayushman, Driving Licence, and ABHA.
- Front + Back upload panels with independent crop selection.
- A4 print-ready PDF output with 1, 2, 4, or 6 copies.
- Standard card print size retained at 85.6 × 54 mm.

## Changed
- Document Studio replaces simple image-only printing with a PDF/image crop workflow similar to the older Aadhaar drag crop tool.
- Cache version updated to v37.2-alpha.

## Apps Script
- No Apps Script update required for this version.
