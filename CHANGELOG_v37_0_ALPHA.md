# Smart Photo Toolkit Pro v37.0 Alpha – Image Studio Rewrite

## Added
- New final studio architecture: Image Studio, Document Studio, PDF Studio.
- Passport Photo moved inside Image Studio conceptually.
- Image Studio landing with Passport Studio, Compressor, Resize, Converter, Name & Date, and Photo Size Library.
- Photo Size Library with Passport, 2x2, Visa, OCI, and Custom Size foundations.
- Document Studio alpha with Aadhaar, Voter ID, PAN, Ayushman, Driving Licence, and ABHA card print support.
- Universal card print engine for image-based front/back card printing.
- PDF Studio grouped page with PDF Resize active and merge/split/rotate planned.
- Cache-busting references for v37.0 alpha.

- Main sidebar now focuses on studio-level navigation instead of many single-purpose buttons.
- Home page now shows the v37 final architecture.
