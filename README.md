# Smart Photo Toolkit Pro v38 Enterprise UI Rewrite

This package is a real GitHub-ready frontend rewrite based on the user's latest V36 project base.

## Highlights
- Enterprise-style topbar + sidebar UI
- Document Studio layout inspired by the provided reference screenshot
- Aadhaar, PAN, Voter ID, Ayushman, ABHA and Driving Licence document tiles
- Front / Back image upload support
- Full-page PDF upload and crop support
- 8-handle selectable crop box
- A4 top margin minimized to 3mm
- Front/back gap minimized to 3mm for fold/lamination
- Passport Photo crop uses the same 8-handle crop engine
- Dynamic top-right user profile menu

## Apps Script
Apps Script update is not required for this UI update. The latest backend is included as ALL_IN_ONE_Code_v38.gs only for reference.
